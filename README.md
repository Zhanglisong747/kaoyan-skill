# 🎯 考研督学教练 · Kaoyan Study Coach

## 🛠️ 全部 10 个工具

### 1️⃣ `/kaoyan plan` — 智能学习规划

根据用户考试时间、当前水平、每日可用时间，自动生成**分阶段个性化学习计划**，覆盖从基础补强到考前冲刺四个阶段。

```
📋 考研英语 · 强化阶段学习计划
倒计时 182 天 | 基础评估：阅读 14/40 · 目标 65+
┌──────────┬──────────┬──────────┬────────────┐
│ 阶段      │ 时间      │ 重点任务  │ 每日时长    │
├──────────┼──────────┼──────────┼────────────┤
│ 基础补强  │ 6-7月     │ 单词+长难句│ 2h/d       │
│ 强化训练  │ 8-9月     │ 真题精刷   │ 2.5h/d     │
│ 冲刺提分  │ 10-11月   │ 套卷模考   │ 3h/d       │
│ 考前调整  │ 12月      │ 查漏补缺   │ 2h/d       │
└──────────┴──────────┴──────────┴────────────┘
```

**用法**：`/kaoyan plan` 或 `/kaoyan plan 英语 周计划`

**参考算法**：`references/study-plan.md` — 四阶段模型、每日时间分配矩阵、目标分解算法、计划调整触发条件

### 2️⃣ `/kaoyan checkin` — 每日打卡

记录每日学习内容，自动计算完成率，更新连续打卡天数，识别学习模式，智能提醒遗忘曲线复习。

```
📅 连续打卡第 15 天 🔥
┌────────────────┬────────┬──────────┬────────────────┐
│ 项目            │ 计划    │ 实际完成  │ 完成率          │
├────────────────┼────────┼──────────┼────────────────┤
│ 英语阅读 1篇     │ 1篇    │ 1篇      │ ██████████ 100%│
│ 单词 3个list     │ 150个  │ 150个    │ ██████████ 100%│
│ 长难句 5句       │ 5句    │ 5句      │ ██████████ 100%│
└────────────────┴────────┴──────────┴────────────────┘

📊 本周趋势：阅读 ████████░░ 80% · 单词 █████████░ 90%
```

**用法**：直接输入学习内容，我会自动解析
```
/kaoyan checkin
今天做了2010年Text4，对了4题错了1题，背了List 10-12
```

**智能督学规则**：
- 连续 2 天未打卡 → 主动询问是否需要调整计划
- 连续 3 天完成率 < 50% → 建议减量或换策略
- 某题型连续 3 次正确率 < 50% → 推送专项练习建议
- 距离考试 < 30 天 → 自动切换冲刺模式

### 3️⃣ `/kaoyan review` — 阅读真题精析

对做完的一篇阅读进行全方位精析：错因归因、题型分类、技巧总结、生词提取、同类巩固推荐。

```
📖 考研英语二 · 2010 Text2 精析
正确率：3/5 (60%) | 用时：18min | 难度：★★★★☆

❌ 第2题 (细节题) — 选B 正答:D
   ├ 错因：定位偏差（定位到第三段而非第二段）
   ├ 技巧：细节题先定位再判断，定位词选名词短语优先
   └ 巩固：建议加练 2011T1Q3 · 2012T2Q1

❌ 第4题 (态度题) — 选C 正答:A
   ├ 错因：态度误判（被段尾负面词误导）
   ├ 技巧：but 转折后才是作者真实态度
   └ 巩固：建议加练 2010T3Q5 · 2011T4Q2

📊 能力雷达：细节题 80% · 态度题 40% ← 薄弱 · 主旨题 60% · 猜词题 90%
```

**用法**：`/kaoyan review 2010 Text2` 或做完题后提交答案

**参考**：`references/reading-analysis.md` — 5 大题型技巧库、干扰项 5 大套路、做题时间分配表

### 4️⃣ `/kaoyan errors` — 错题本分析

管理错题数据库，按题型/错因/时间段统计，可视化薄弱环节，推荐针对性练习。

```
📊 错题统计分析 · 总 47 题 | 12 篇 | 2010T1→2014T3

按题型分布：
细节题    ████████████████░░░░ 18题 (38%)
态度题    ██████████████░░░░░░ 14题 (30%) ← 重点
主旨题    ████████░░░░░░░░░░░░ 8题 (17%)
推断题    ██████░░░░░░░░░░░░░░ 6题 (13%)
猜词题    █░░░░░░░░░░░░░░░░░░░ 1题 (2%)  ✓ 强项

按错因归类：
定位偏差 ████████████░░░░░░ 32% · 态度误判 ██████████░░░░░░ 26%
选项混淆 ████████░░░░░░░░░░ 21% · 主旨偏离 █████░░░░░░░░░░░░ 13%
```

**用法**：`/kaoyan errors`（总览）· `/kaoyan errors list 态度题`（筛选）· `/kaoyan errors export`（导出）

**参考**：`references/error-analysis.md` — 5 类 MECE 错因定义、错题数据库结构、针对性练习推荐表

### 5️⃣ `/kaoyan vocab` — 艾宾浩斯单词复习

基于 **SM-2 间隔重复算法**，自动调度每日新学和复习，管理生词库，跟踪掌握率趋势。

```
📚 单词复习 · 总词库 1,247 | 已掌握 526 | 学习中 721

今日复习计划（遗忘曲线调度）：
🔄 第1次复习（1天后）: List 10-12 (48词)
🔄 第2次复习（3天后）: List 7-9  (42词)
🔄 第3次复习（7天后）: List 4-6  (45词)
🔄 第4次复习（15天后）: List 1-3 (40词)

📝 今日新学：List 13 (25词) · 预计 45min
```

**用法**：`/kaoyan vocab new`（新学）· `/kaoyan vocab review`（复习）· `/kaoyan vocab status`（状态）· `/kaoyan vocab export`（导出）

**参考**：`references/vocab-system.md` · `references/spaced-repetition.md`

### 8️⃣ `/kaoyan sentences` — 长难句拆解

分析长难句结构：成分解析、语法点提取、翻译技巧。

```
📝 原句：This trend, which some believe is being driven by
the growing power of social media, has significant implications...

📐 结构：
[主句] This trend has significant implications
   ├ [插入定语从句] which some believe is being driven by...
   └ [介词宾语从句] how we understand public opinion

🏗 成分：主语→谓语→宾语→定语（介词短语）→插入（非限制性定语从句）
🔑 考点：非限制性定语从句 · 介词+宾语从句 · 被动语态
```

**用法**：`/kaoyan sentences [粘贴句子]`

## 🧠 核心算法

### 错因归因模型（MECE）

```
错因编码    类型        占比参考
───────── ────────── ─────────
LOC       定位偏差     30-40%  ← 最常见的错误
ATT       态度误判     15-25%
CONF      选项混淆     15-25%
MAIN      主旨偏离     10-15%
VOC       词汇障碍      5-10%
```

### 艾宾浩斯间隔调度（SM-2）

| 复习次数 | 间隔 | 时间点 |
|---------|------|-------|
| 第1次 | 1天后 | Day 1 |
| 第2次 | 3天后 | Day 3 |
| 第3次 | 7天后 | Day 7 |
| 第4次 | 15天后 | Day 15 |
| 第5次 | 30天后 | Day 30 |

### 学习风格自适应

| 行为特征 | 推断风格 | 策略 |
|---------|---------|------|
| 经常超额完成 | 自律型 | 适当加量 |
| 完不成但连续学 | 坚持型 | 减量保质量 |
| 三天打鱼两天晒网 | 波动型 | 先养成习惯 |
| 正确率高但慢 | 谨慎型 | 限时提速 |
| 正确率低但快 | 冲动型 | 强制检查 |
