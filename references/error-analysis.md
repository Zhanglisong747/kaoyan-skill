# 错题分析系统

## 归因模型

所有错题归入以下5类（MECE原则，互斥且穷尽）：

| 错因编码 | 错因类型 | 定义 | 占比参考 |
|---------|---------|------|---------|
| LOC | 定位偏差 | 关键词定位到错误段落/句子 | 30-40% |
| ATT | 态度误判 | 把作者态度/观点判断反了 | 15-25% |
| CONF | 选项混淆 | 在两个选项间犹豫选错 | 15-25% |
| MAIN | 主旨偏离 | 没抓住文章/段落主旨 | 10-15% |
| VOC | 词汇障碍 | 关键生词导致理解偏差 | 5-10% |

## 错题数据库结构

每道错题记录以下字段：
```
{
  "id": "2010T2-Q3",
  "year": 2010,
  "text": 2,
  "question": 3,
  "type": "细节题",
  "errorCode": "LOC",
  "selected": "B",
  "correct": "D",
  "difficulty": 4,
  "keySentence": "Paragraph 2, Line 3",
  "reason": "定位到第三段instead of第二段",
  "skill": "细节题先定位再判断，定位词选名词短语优先",
  "reviewDate": "2026-06-21"
}
```

## 针对性练习推荐

基于错因，推荐针对性训练：

| 错因 | 推荐练习 | 训练方法 |
|------|---------|---------|
| 定位偏差 | 同义替换专项 | 每天10组同义替换配对练习 |
| 态度误判 | 态度词库积累 | 整理正/负/中立态度词各30个 |
| 选项混淆 | 干扰项特征识别 | 总结5类干扰项套路 |
| 主旨偏离 | 首尾句串读法 | 每段只读首尾句，3分钟内串完 |
| 词汇障碍 | 真题语境背词 | 把生词放回原句背 |
