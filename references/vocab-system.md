# 单词记忆系统

## 艾宾浩斯间隔复习调度

背词后的复习时间点：

| 复习次数 | 间隔时间 | 时间点（以背词日为Day 0） |
|---------|---------|------------------------|
| 第1次 | 1天后 | Day 1 |
| 第2次 | 3天后 | Day 3 |
| 第3次 | 7天后 | Day 7 |
| 第4次 | 15天后 | Day 15 |
| 第5次 | 30天后 | Day 30 |

## 每日复习计算公式

```
今日复习词数 = Σ(第i天新学词数, i ∈ 需复习的天数)
```

例如：
- 今天第15天 → 需复习 Day14(1天后) + Day12(3天后) + Day8(7天后) + Day0(15天后) 的新学词

## 单词掌握度评级

| 级别 | 标准 | 复习间隔 |
|------|------|---------|
| ❌ 不认识 | 完全没印象 | 立即再学 |
| ⚠️ 模糊 | 3-5秒才想起 | 缩短间隔50% |
| ✅ 认识 | 1-2秒反应出 | 按标准间隔 |
| 🏆 掌握 | 瞬间反应+能造句 | 间隔翻倍 |
| 🎉 已归档 | 连续5次✅ | 不再出现 |

## 真题高频词优先级

```
第一梯队（出现≥20次）： 必背，阅读基石
第二梯队（出现10-19次）：核心词汇，影响理解
第三梯队（出现5-9次）： 常见词，拓展
第四梯队（出现1-4次）： 随缘，有印象即可
```
