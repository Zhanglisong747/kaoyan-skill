# 阅读精析方法论

## 五大题型技巧库

### 1. 细节题（占比最高 ~40%）

**识别标志**：According to the passage, ... / Which of the following is true? / ...because...

**三步法**：
```
① 题干定位 → 圈关键词（名词短语优先）
② 回文扫描 → 找同义替换
③ 选项匹配 → 排除干扰项
```

**干扰项套路**：
| 类型 | 特征 | 排除方法 |
|------|------|---------|
| 信息拼凑 | 文中出现过的词拼一起 | 检查逻辑关系 |
| 反向干扰 | 与原文意思相反 | 注意否定词 |
| 无中生有 | 原文没提 | 回文确认 |
| 范围错位 | 扩大或缩小范围 | 对比限定词 |
| 偷换概念 | 换了关键词 | 精确比对 |

### 2. 态度题

**识别标志**：attitude / tone / feel about / according to the author

**方法论**：
```
① 找态度信号词（but/however → 转折后）
② 排除客观类选项（objective/neutral一般不选）
③ 确认是谁的态度（作者 vs 文中某人）
```

**态度词库**：
| 正向 | 负向 | 中立 | 干扰 |
|------|------|------|------|
| approving | critical | objective | indifferent (一般不选) |
| supportive | skeptical | neutral | biased (一般不选) |
| optimistic | pessimistic | impartial | puzzled |
| favorable | opposed | balanced | |

### 3. 主旨题

**识别标志**：best title / main idea / mainly about / purpose

**方法论**：
```
① 串读各段首尾句
② 找全文中心词（高频出现的名词/概念）
③ 排除只在某一段出现的选项
```

### 4. 猜词题

**识别标志**：The word/phrase "..." probably means

**方法论**：
```
① 定位上下句逻辑（并列/转折/因果/解释）
② 代入验证
③ 排除字面意思（一般不选原词字面义）
```

### 5. 推断题

**识别标志**：imply / infer / suggest / conclude / learn from

**方法论**：
```
① 找到推理依据句
② 不超过原文信息半步
③ 排除原文直接说的（推理不是复述）
```

## 做题时间分配

| 步骤 | 时间 | 说明 |
|------|------|------|
| 看题干 | 1min | 只看题干不看选项 |
| 通读文章 | 4min | 重点读首尾句+转折处 |
| 做题 | 8min | 5题，平均1.6min/题 |
| 检查 | 2min | 不确定的题再看一次 |
| **总计** | **15min** | **一篇阅读** |

## 文章类型与应对

| 类型 | 特点 | 应对策略 |
|------|------|---------|
| 社会科学 | 论点+论据结构清晰 | 抓首段主旨和段首句 |
| 自然科学 | 实验过程+结论 | 结论远比过程重要 |
| 商业经济 | 现象+原因+影响 | 关注因果逻辑链 |
| 文化教育 | 比较/对比结构 | 区分新旧观点 |
