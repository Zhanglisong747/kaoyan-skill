# 学习计划系统

## 四阶段模型

考研英语标准四阶段划分（以12月考试为终点）：

```
倒计时 180+ 天 → 基础补强期
倒计时 120-180 天 → 强化训练期
倒计时 60-120 天 → 冲刺提分期
倒计时 <60 天 → 考前调整期
```

## 每日时间分配建议

| 每日可用时间 | 阅读 | 单词 | 长难句 | 其他 |
|------------|------|------|--------|------|
| 1-2h | 30min | 30min | 15min | 15min |
| 2-3h | 45min | 40min | 20min | 15min |
| 3-4h | 60min | 45min | 25min | 30min |
| 4h+ | 90min | 60min | 30min | 60min |

## 目标分解算法

```
总目标分数 → 各题型目标分 → 正确率要求 → 每日任务量 → 阶段检查点
```

例如：目标65分
- 阅读：28/40 (70%)
- 完形：6/10 (60%)
- 新题型：8/10 (80%)
- 翻译：8/15 (53%)
- 作文：15/25 (60%)

## 计划调整触发条件

- ⚠️ 连续3天完成率<60% → 减量30%
- ⚠️ 某题型正确率连续5篇不升反降 → 换方法
- ⚠️ 每日耗时超预期>30% → 拆小任务粒度
- ✅ 连续7天完成率>90% → 适当加量
